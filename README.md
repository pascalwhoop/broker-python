# PowerTAC Python Adapter to broker-adapter

## Installation

```
pip install grpc
python main.py
```

## Code Generation
run the shell script in the root of the project. but first run `pip3 install grpcio-tools` and make sure you have python3 and pip3 installed and in your PATH.
